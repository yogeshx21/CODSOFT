#  Student Grade Calculator

## Description

Student Grade Calculator is a Java console-based application developed as part of the CODSOFT Java Programming Internship. This program allows users to enter the student's name, the number of subjects, and marks obtained in each subject (out of 100). It automatically calculates the total marks, average percentage, assigns a grade based on the percentage, identifies the highest and lowest marks, and determines whether the student has passed or failed. The application provides a simple, accurate, and user-friendly way to evaluate a student's academic performance.

# Features
 Student name input
 Multiple subject support
 Total marks calculation
 Average percentage calculation
 Grade calculation
 Highest marks display
 Lowest marks display
 Pass/Fail result
 User-friendly console interface

# Grade Criteria
| Percentage | Grade |
|------------|-------|
| 90% and Above | A+ |
| 80% - 89% | A |
| 70% - 79% | B |
| 60% - 69% | C |
| Below 60% | D |

A student is considered **FAIL** if marks in any subject are below **35**.

# How to Run
1. Install JDK 8 or above.
2. Download or clone this repository.
3. Open the project in VS Code or any Java IDE.
4. Compile the program:
   ```bash
   javac StudentGrade.java
   ```
5. Run the program:
   ```bash
   java StudentGrade
      ```
