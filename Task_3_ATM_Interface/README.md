#  ATM Interface

# Description

ATM Interface is a Java console-based application developed as part of the CODSOFT Java Programming Internship. The project simulates the basic functionality of an Automated Teller Machine (ATM). It allows users to check their account balance, deposit money, withdraw money, and validates transactions to ensure sufficient balance and valid input. The application uses separate classes for the ATM and the Bank Account, demonstrating the concept of Object-Oriented Programming (OOP) in Java.


# Features

 Check account balance
 Deposit money
 Withdraw money
 Input validation
 Prevents withdrawal when balance is insufficient
 Prevents invalid deposit or withdrawal amounts
 Menu-driven console interface
 Uses separate ATM and BankAccount classes

# Functionalities
 Check current account balance
 Deposit money into the account
 Withdraw money from the account
 Display success and error messages
 Exit the application safely

#How to Run
1. Install JDK 8 or above.
2. Download or clone this repository.
3. Open the project in VS Code or any Java IDE.
4. Compile the program:
   ```bash
   javac ATM.java
   ```
5. Run the program:
   ```bash
   java ATM
   ```
